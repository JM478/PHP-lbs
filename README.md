***Tankrechner***  
Das Projekt berechnet die Tankkosten. Man gibt die Anzahl der Liter und den Preis ein, und es zeigt die Kosten an.

# Installation
> XAMPP
1. Gehe ins htdocs Verzeichnis
2. Öffne den Terminal
3. Führe aus `git clone https://github.com/JM478/PHP-lbs.git`
4. Starte den Webserver durch XAMPP
5. Öffne den Browser und gib als Link `localhost/PHP-lbs/static.php` oder `localhost/PHP-lbs/form.php` ein.

# Verwendung
XAMPP und PHP 8.2.4
